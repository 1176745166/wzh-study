1、jwt是无状态的,为什么要放入redis,不用redis不行吗？
答：
①安全考虑：如果仅依赖 JWT，当用户登出或修改密码时，无法立即使 JWT 失效，因为 JWT 在有效期内始终有效。通过在 Redis 中存储用户信息，可以在用户登出时删除 Redis 中的记录，从而实现立即失效。
②控制令牌生命周期：通过 Redis 存储用户信息，可以更灵活地控制令牌的生命周期，例如在用户活动时自动刷新令牌有效期。
③防止令牌滥用：在 Redis 中存储令牌信息可以防止令牌被恶意使用，例如通过令牌黑名单机制。
④性能考虑：虽然 JWT 无状态，但如果需要频繁访问用户信息，每次都解析 JWT 可能会影响性能。通过 Redis 缓存用户信息，可以提高访问速度。
